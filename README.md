## Features

* uses:
  * React (create-react-app)
  * MobX

## Installation

* `git clone git@github.com:rwieruch/react-mobx-hackernews.git`
* `cd react-mobx-hackernews`
* `npm install`
* `npm start`
* visit http://localhost:3000/
